# ModelSnatch
FFXI Addon - Displays model # of PC and NPC in game.

        "=== ModelSnatch Help ==="

        "  //snatch npc     - Snatch models without names (for NPCs)"
        "  //snatch export  - Export snatched data to DressUp text file"
        "  //snatch hide    - Hide the UI overlay"
        "  //snatch         - Toggle this help menu"
